V26 SEO / Google Search Fix
===========================

تمت إضافة ملفات وتحسينات SEO حتى يظهر الموقع بشكل أفضل عند البحث عن:
- مدارس العصر الذهبي
- العصر الذهبي
- MGA
- mga
- Modern Golden Age School

الملفات المعدلة/المضافة:
1) index.html
   - عنوان SEO واضح
   - description
   - keywords
   - robots meta
   - canonical
   - Open Graph
   - Twitter meta
   - Schema.org School JSON-LD
   - محتوى HTML أولي قابل للقراءة من محركات البحث قبل تحميل React

2) public/robots.txt
   - السماح بالفهرسة
   - ربط sitemap.xml

3) public/sitemap.xml
   - خريطة موقع للصفحة الرئيسية وصفحة الدخول

مهم:
- الظهور في Google ليس فوريًا. يحتاج من ساعات إلى أيام، وأحيانًا أكثر.
- الأفضل إضافة الموقع في Google Search Console وطلب الفهرسة يدويًا.
- عند تغيير رابط Netlify النهائي، عدّل الرابط داخل index.html و robots.txt و sitemap.xml.
